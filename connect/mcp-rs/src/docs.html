Configure your client

Claude Code / Codex / any MCP-aware CLI

Add an HTTP MCP server pointing at this host:

{
  "mcpServers": {
    "ohd": {
      "url": "https://mcp.ohd.dev/mcp",
      "transport": "http"
    }
  }
}

(Some clients still expect stdio; run ohd-mcp-rs stdio locally for those — see the repo README.)

Auth

v1 ships single-user — the deployed instance opens one storage file and serves it without bearer auth. Multi-user routing tracks against api.ohd.dev's profile model and lands in the next iteration. For now: don't point untrusted clients at mcp.ohd.dev.
